== Description ==

The Weight Loss Club WordPress Theme is a premium, versatile theme designed specifically for those seeking to create a supportive online community for weight loss. Whether you&#39;re running a weight loss program, fitness club, or wellness group, this theme offers all the features needed to build an engaging platform for your members. It’s ideal for weight loss support groups, personalized weight loss plans, or online weight loss clubs. The theme enables users to track their weight loss journey, set goals, and participate in challenges or transformations with ease. The design is clean, user-friendly, and fully responsive, ensuring an optimal experience across all devices. With dedicated sections for weight loss tips, success stories, meal plans, and fitness tracking, it offers a comprehensive space for users to engage and stay motivated. The theme includes customizable features like a weight loss tracker, virtual fitness challenges, weekly weigh-ins, and coaching support, all of which foster accountability and group motivation. Additionally, it’s equipped with powerful tools for managing member interactions, such as forums, group coaching, and progress tracking. Designed to support a range of weight loss methods, from low-carb and intermittent fasting groups to holistic and sustainable weight loss approaches, this theme helps users stay on track and reach their fitness and health goals.

== Homepage Installation ==

Step-1. Go to Dashboard -- Pages -- Add new page -- Add templates (Home Page) -- Publish it.
Step-2. Go to Dashboard -- Settings -- Reading -- Click on static page -- Select home page -- Save changes.

Slider Section
Step-1. Go to Dashboard -- Posts -- Categories -- Add Name -- Add New Category.
Step-2. Go to Dashboard -- Posts -- Add New Post -- Add title, content, and featured image -- Publish it.
Step-3. Go to Dashboard -- Appearnace -- Customize -- Theme option panel -- Manage Slider Section -- Check To Enable This Section -- Select Category to display Latest Post -- Add Sub-Title -- Add Button Text -- Select Banner Background Image -- Publish it.

Trainer Section
Step-1. Go to Dashboard -- Posts -- Categories -- Add Name -- Add New Category.
Step-2. Go to Dashboard -- Posts -- Add New Post -- Add title, content, and featured image -- Publish it.
Step-3. Go to Dashboard -- Appearnace -- Customize -- Theme option panel -- Manage Trainers Section -- Check To Enable Section -- Fill the Trainer information  -- Publish it.

Footer Section
Step-1. Go to Dashboard -- Appearnace -- Customize -- Theme option panel -- Manage Footer Section -- Add copyright text -- Publish it.
